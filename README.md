# Minesweeper
This version of Minesweeper is the classic rules and has 3 levels of difficulty including:
* Easy, 9x9, 10 mines
* Medium, 16x16, 32 mines
* Hard, 16x30, 60 mines

## Technoligies Used
* C#
* .NET 8
* WinForms
